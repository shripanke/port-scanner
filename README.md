# Port Scanner

## Description
A Python-based port scanner that scans a target IP address to detect open ports. This project demonstrates basic networking and cybersecurity concepts using socket programming.

## Features
- Scans ports from 1 to 1024
- Detects open ports
- Simple and beginner-friendly

## How to Run
1. Install Python
2. Open terminal in project folder
3. Run:
   python scanner.py
4. Enter target IP (example: 127.0.0.1)

## Example Output
Port 80 is open
Port 443 is open
